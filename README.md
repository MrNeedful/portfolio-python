# 🤿 Diving Log — Registro de Inmersiones

Un registro personal de inmersiones de buceo hecho en Python.

## ¿Qué hace?
Permite registrar y consultar inmersiones de buceo con sus datos principales:
sitio, ubicación, profundidad, duración, temperatura y visibilidad.

## Funcionalidades
- Agregar nuevas inmersiones
- Ver la inmersión más profunda
- Ver la inmersión menos profunda
- Ver la inmersión de mayor duración
- Filtrar inmersiones por ubicación
- Ver todas las inmersiones

## Requisitos
- Python 3.10 o superior

## ¿Cómo instalarlo?
```bash
git clone https://github.com/MrNeedful/portfolio-python.git
cd portfolio-python/buceo
```

## ¿Cómo ejecutarlo?
```bash
python diving.py
```
