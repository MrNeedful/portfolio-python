# 🤿 Diving Log — Registro de Inmersiones

Un registro personal de inmersiones de buceo hecho en Python.

## ¿Qué hace?
Permite guardar inmersiones con sus datos principales: sitio, ubicación, 
profundidad, duración, temperatura y visibilidad.

## ¿Cómo ejecutarlo?
1. Cloná el repositorio
2. Abrí `buceo/diving.py` en VS Code
3. Dale al play — el programa te va a pedir los datos de la inmersión

## Tecnologías
- Python 3
